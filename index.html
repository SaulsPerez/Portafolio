Visual Aids Display System

The project is developed in Ionic with Angular, with a database provided by a CMS (Strapi). Consists of a CRUD system which stores the plant's visual aids in order to reduce paper usage. These visual aids can be categorized by area and are equipped with a QR code which, upon scanning, displays the visual aids for that specific area.

The Application of Overtime Hours

The project is developed using Ionic with Angular, with a SQL database and information obtained through an API. Consists of a system for registering overtime hours grouped by departments, in which the user can record the time of their employees and indicate the reason. Once the record is saved, a notification is sent to approver users to accept or reject the overtime registration. The system includes graphs to observe the amount of hours per reasons.

Scoring Display System

The project is developed using Ionic with Angular, with a SQL database and information obtained through an API. The web page displays information about workers in a scorecard, allowing users to review employee scores and categories in a clear and simple way. This solution improves efficiency and transparency in employee performance evaluation within the company.
